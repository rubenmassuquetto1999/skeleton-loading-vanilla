# Skeleton Loading Vanilla 🦴

Este projeto é uma implementação de **Skeleton Loading** (telas de carregamento esqueleto) utilizando apenas tecnologias web nativas (HTML, CSS e JavaScript puro). O objetivo é melhorar a experiência do usuário (UX) ao simular o carregamento de dados antes que eles sejam exibidos na tela.

## 🚀 Sobre o Projeto

O Skeleton Loading é uma técnica de design que exibe uma versão simplificada da interface (geralmente blocos cinzas pulsantes) enquanto o conteúdo real está sendo processado ou buscado em uma API. Isso reduz a percepção de tempo de espera e evita mudanças bruscas de layout (Cumulative Layout Shift).

### Principais Funcionalidades:
- **Efeito Shimmer:** Animação de brilho que percorre os elementos de esqueleto.
- **Simulação de Delay:** Uso de temporizadores para demonstrar a transição entre o estado de carregamento e o conteúdo final.
- **Design Responsivo:** Interface moderna adaptada para diferentes tamanhos de tela.
- **Feedback Visual:** O botão altera seu estado e texto durante o carregamento.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** - Flexbox para alinhamento.
  - Keyframes para as animações `shimmer` e `fadeIn`.
  - Background-clip para texto com gradiente.
- **JavaScript (Vanilla):** Manipulação de DOM, renderização dinâmica de elementos e controle de estado.
